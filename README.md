# Social Media App API

This is a Node.js API for a social media app, which allows users to perform various actions such as creating posts, liking posts, and following other users.

## Features

- User authentication (register, login, logout)
- Create, read, update, and delete posts
- Like and dislike posts
- Follow and unfollow other users
- Retrieve user profiles and posts
- Error handling and validation

## Technologies Used

- Node.js
- Express.js
- MongoDB (with Mongoose)
- JSON Web Tokens (JWT) for authentication
- bcrypt.js for password hashing
- Other required dependencies (check package.json)
